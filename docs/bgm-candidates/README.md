## 试听顺序建议

按这个顺序听，差异最明显：

1. `bgm-2-lofi-hip-hop-morning-coffee.mp3` — 最 "有节奏"，鼓+钢琴+vinyl
2. `bgm-1-cinematic-piano-lightless-dawn.mp3` — 最 "电影感"，重复钢琴 motif + pad
3. `bgm-5-minimalist-piano-crinoline-dreams.mp3` — 最 "克制有 mood"，piano + 弦乐 + bass
4. `bgm-3-electronic-ambient-floating-cities.mp3` — 最 "科技感"，电子 pulse + arpeggio
5. `bgm-4-cinematic-build-hitman.mp3` — 最 "戏剧化"，从安静 swell 到 climax

## 候选详情

### 1. Cinematic minimal piano with hook

**`bgm-1-cinematic-piano-lightless-dawn.mp3`**

- **曲目**：Lightless Dawn
- **作者**：Kevin MacLeod
- **来源**：https://incompetech.com/music/royalty-free/index.html?isrc=USUAN1100655
- **License**：CC BY 4.0（**需署名** — 在 README/credits 写 "Music: 'Lightless Dawn' Kevin MacLeod (incompetech.com) Licensed under Creative Commons: By Attribution 4.0 License http://creativecommons.org/licenses/by/4.0/"）
- **原曲时长**：6:19 / 320kbps mp3
- **Preview 段**：原曲 145s → 175s（最强的 piano motif 段，pad 已建立）
- **风格匹配**：Max Richter / Ólafur Arnalds 风。低频 pad + 间歇性 piano motif。
- **对位 Pace demo 5 反差点**：
  - 0-10s（pad 起）= 用户打开 Pace 静态界面
  - 10-20s（piano motif 第一次出）= "我现在在干啥" 触发
  - 20-30s（pad swell）= mentor 输出展开
- **推荐度**：★★★★★ — 最像 "Linear / Vision Pro / Apple WWDC keynote" 风格，和 Pace 工具气质最贴

---

### 2. Lo-fi hip-hop

**`bgm-2-lofi-hip-hop-morning-coffee.mp3`**

- **曲目**：Morning Coffee
- **作者**：HoliznaCC0
- **来源**：https://freemusicarchive.org/music/holiznacc0/lo-fi-and-chill/morning-coffee/
- **License**：**CC0 / Public Domain**（无需署名，完全自由商用）
- **原曲时长**：3:12 / 320kbps mp3
- **Preview 段**：原曲 20s → 50s（鼓已加入 + 钢琴 hook 稳定段）
- **风格匹配**：Nujabes / lo-fi hip-hop 经典套路。boom-bap 鼓 + 复古钢琴 sample + vinyl crackle。
- **对位 Pace demo 5 反差点**：
  - 鼓的稳定 4/4 = 视频画面切换有 anchor
  - 钢琴 sample 循环 = 让 30s/108s 反差点不显得突兀
  - 整体气质偏 "周末写代码 / 晚上加班"，**和 Pace 给职场人陪伴感最匹配的一首**
- **推荐度**：★★★★☆ — 如果 Pace 想走 "温柔陪伴" 路线（不是 "Apple 工具" 路线），选这首

### 3. Electronic ambient with pulse

**`bgm-3-electronic-ambient-floating-cities.mp3`**

- **曲目**：Floating Cities
- **作者**：Kevin MacLeod
- **来源**：https://incompetech.com/music/royalty-free/index.html?keywords=floating+cities
- **License**：CC BY 4.0（需署名）
- **原曲时长**：3:04 / 256kbps mp3
- **Preview 段**：原曲 90s → 120s（arpeggio + 暖 pad 完整层叠段）
- **风格匹配**：Tycho / Bonobo 风。软 kick 暗示 + arpeggio synth + 暖 pad。

- **推荐度**：★★★☆☆ — 安全选择，"科技产品视频"标配，但不一定让人记住

### 4. Cinematic build with crescendo

**`bgm-4-cinematic-build-hitman.mp3`**

- **曲目**：Hitman
- **作者**：Kevin MacLeod
- **来源**：https://incompetech.com/music/royalty-free/index.html?isrc=USUAN1300013
- **License**：CC BY 4.0（需署名）
- **原曲时长**：3:21 / 320kbps mp3
- **Preview 段**：原曲 50s → 80s（**从 -28dB 安静段一路 swell 到 -12dB climax**，30s 内有强 build）
- **风格匹配**：Hans Zimmer minimal 风。起点克制，慢慢 layer 上来。

  - 30s 内有 8dB swing → 视觉反差点能被音乐"撑起来"
- **推荐度**：★★★★☆ — 如果 demo 视频结尾有"翻转 / 揭示"的戏剧时刻，这首是最佳选择。但日常使用可能 too much。

### 5. Minimalist piano + sub-bass drone

**`bgm-5-minimalist-piano-crinoline-dreams.mp3`**

- **曲目**：Crinoline Dreams
- **作者**：Kevin MacLeod
- **来源**：https://incompetech.com/music/royalty-free/index.html?keywords=crinoline+dreams (Bandcamp: https://kevinmacleod.bandcamp.com/track/crinoline-dreams)
- **License**：CC BY 4.0（需署名）
- **原曲时长**：4:06 / 256kbps mp3
- **Preview 段**：原曲 30s → 60s（piano + viola + bass 全部就位的稳定段）
- **风格匹配**：Nils Frahm 风。极简钢琴 + 弦乐 drone + double bass 低频。

  - 但 piano motif 重复，**有 hook 可记**（解决用户"太淡没特色"的核心抱怨）
- **推荐度**：★★★★☆ — "克制但有特色" 的最优解。比 Lightless Dawn 更现代，比 Hitman 更日常。

## 我的推荐（按 demo 用例排序）

1. **Pace 想走 Apple/Linear 工具风** → `bgm-1` Lightless Dawn
2. **Pace 想走 "陪伴打工人" 路线** → `bgm-2` Morning Coffee
3. **Pace 视频有戏剧反差点** → `bgm-4` Hitman（替代或拼到结尾 7s）
4. **想要"克制 + 有 hook"折中** → `bgm-5` Crinoline Dreams
5. **保守安全选** → `bgm-3` Floating Cities

**单选首推**：`bgm-1` Lightless Dawn。Pace 是"PMP mentor"工具，气质是冷静 + 思辨 + 引导，钢琴 motif + 低频 pad 的组合最贴合产品调性，且 Linear/Vision Pro/Apple 这类工具视频常用同类型 BGM，用户记忆点会自动对齐到"高级工具"印象。
