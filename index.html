main 上裸改 3 小时后，它问你一句话。

不是提醒你建分支——是让你停一下：你在哪个回合、谁等着你的下游、要不要先同步。

本地优先 · 你的电脑

02 · 它在看你跟 cc 说了啥

先翻你今天的对话，
再开口。

「我刚改了 IPC race bug，下一步该干啥？」

↓
在看你 ~/projects/pace 翻过 5 commits · 47 行 diff 读过 cc 8 轮对话 才开口 29 秒

不上传任何字节 · 只在你电脑里

03 · 反定义

它不是—

  • 不是 task tracker / Jira / Linear / 看板 → 它看你今天已经做了什么
  • 不是 ChatGPT 替代品 → 每一句都基于你今天真实改过的代码
  • 不是 IDE 插件 / cc sidebar → 独立 桌面 dock，关上窗就消失

04 · 它开口时长这样

不是问你今天怎么样，
是先翻完证据再说话。

阶段

你这周在做收尾活。

找谁

这事得 Tom 点头。

话怎么开

「Tom，我这周一直在收 export 这块的尾，error code 的语义你那边比我清楚——抽你 15 分钟对一下，我再封 PR。」

基于你今天 5 个 commit · 8 轮 cc 对话

Tom · 这事得他点头

它是 mentor。 它读你的 cc。 它在你电脑里。 它告诉你下一步找谁、话怎么开。
