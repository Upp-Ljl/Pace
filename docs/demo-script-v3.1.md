# Pace Demo Video Script v3.1 — ≤2min no-VO 用户化语言版

> v3.1 来自反馈：v3 (`demo-script-v3.md`) 的 7 个视觉创新点都对，但有两个用户体感问题。
>
> 1. **PMP 没幕后化够**：v3 旁白与凭据角注里出现 `PMBOK 6 · § Communications` / `Executing × 整合` / `RACI: R` / `Stakeholder` 等内部话。Pace 的真实用户群是**所有用 cc + git 的职场人**（PM、设计、运营、HR、独立开发者等），他们听不懂 PMBOK 术语，看到这些字会觉得"这不是给我的"。PMP 应当只作为**幕后骨架**——视频里只展示输出形态（找谁 / 拍板 / 收尾活），不出现术语。
> 2. **60s 太挤**：v3 4 个反差点（5s bloom / 28s morph / 30s 折叠+黑屏+bloom / 54s 红→紫翻转）被压缩在 60s 里，剪辑师看得到但观众消化不及。用户原话「**2min 内就可以**」——放宽到 ≤120s 让关键反差点真正延伸到剪辑师水平。
>
> v3.1 = v3 的 7 个视觉创新点 100% 保留 + 时长 60s → ≤120s 重新分配 + 所有 PMP 术语换成用户能感知的中文。v3 文档保留不动，本文件独立交付。
